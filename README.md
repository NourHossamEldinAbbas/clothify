Clothify is a web platform built to connect fabric suppliers, factories, workshops, and shops in one organized system.

# Features

- Central ecosystem for all industry roles
- Fast, accurate search for materials, factories, and products
- Professional profiles for shops and factories
- Direct communication and order management

# Project Structure (Prototype)

- index.html — Landing page
- login.html — Sign-in
- register.html — Sign-up with role selection
- clothifyDocumintation.pdf — Full project plan
- useCases.png, classDiagram.jpg, databaseSchema.jpg, clothifyDataflow.jpg — System - - diagrams
